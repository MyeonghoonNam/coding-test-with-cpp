## Compile & Execute

```
// compile
g++ -std=c++14 -Wall "컴파일 파일명(ex: test.cpp)" -o "실행파일명(ex: test.out)"
// compile ex
g++ -std=c++14 -Wall test.cpp -o test.out

// execute
"실행파일명" (ex: test.out)
// execute ex
test.out (window)
./test.out (mac)
```
